# Security Policy

Please report vulnerabilities privately to the repository maintainers.

## Sensitive-data handling

This project must not collect credentials or browser session material. Browser profiles remain local to the user and are ignored by git. If you discover a way for the tool to type credentials, export cookies, or continue through a security prompt automatically, treat it as a security bug.

## Unsupported requests

The project will not support captcha bypass, anti-detection, fingerprint randomization, proxy rotation, automated account creation, or login automation.

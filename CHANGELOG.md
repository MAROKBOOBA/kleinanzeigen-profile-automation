# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project uses semantic versioning while it remains pre-1.0.

## [0.1.0] - 2026-06-14

### Added

- Initial credential-safe CLI for Kleinanzeigen profile automation.
- Manual-login handoff flow that never types credentials.
- Login/security page guard and forbidden sensitive queue keys.
- JSON/CSV/YAML queue loading.
- Review/draft/publish modes with explicit publish confirmation.
- Unit tests, example queue, secret scanner, security policy, and contribution guidelines.
- CI workflow for compile, tests, dry-run, and secret scan.
